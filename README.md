# Cuestionario de Orientación — 1º ESO

Ver instrucciones de despliegue en el chat de Claude.
Resumen rápido:
1. `npm install`
2. Crear proyecto en Supabase, ejecutar sql/schema.sql
3. Copiar .env.example a .env y rellenar (para desarrollo local con `npm run dev`)
4. Subir a GitHub
5. Importar en Vercel, añadir las mismas variables de entorno (server-side sin prefijo VITE_)
6. Deploy
